magic_square = [
    [8, 3, 4],
    [1, 5, 9],
    [6, 7, 2]
]
``````python
# Rows
8 + 3 + 4 = 15
1 + 5 + 9 = 15
6 + 7 + 2 = 15

# Columns
8 + 1 + 6 = 15
3 + 5 + 7 = 15
4 + 9 + 2 = 15

# Diagonals
8 + 5 + 2 = 15
4 + 5 + 6 = 15
```

    To visualize this sqaure, see it like a square. All the rows have same number of elements. Number of elements in row should be equal to number of elements in a column.
